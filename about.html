About Me

Get to know the person behind the code

Background

I'm Rishabh Vishwkarma, a passionate Full Stack Web Developer from Bhopal, Madhya Pradesh. I completed my B.Tech from Rajiv Gandhi Proudyogiki Vishwavidyalaya (RGPV), one of Madhya Pradesh's premier technical universities.

My journey in tech started with curiosity about how websites work, which evolved into a full-fledged passion for building web applications that are both beautiful and functional.

Expertise

I specialize in full-stack web development with expertise across the MERN Stack (MongoDB, Express.js, React, Node.js) and Spring Boot with Thymeleaf.

My frontend skills include HTML5, CSS3, JavaScript, Bootstrap, and Next.js. On the backend, I work with Java, Node.js, PHP, and MySQL databases.

My Approach

I believe in writing clean, maintainable code with a performance-first mindset. Every project I build is:

  • Fully responsive across all devices
  • Optimized for AMOLED displays
  • Built with modern best practices
  • Tested for performance & accessibility
